## 🔬 Featured Projects

### 🖧 [ha-cluster-lab](https://github.com/madhawapathum/ha-cluster-lab)
> PXE-based automated Debian provisioning lab for deploying a high-availability cluster

- Built a **3-node HA cluster** using Pacemaker & Corosync
- Automated OS provisioning via **PXE boot** (dnsmasq, TFTP, HTTP)
- Implemented failover and service availability testing with **Nginx**
- Simulates real-world infrastructure behaviour in a virtualised environment

---

### 📊 [server-perf-stats](https://github.com/madhawapathum/server-perf-stats)
> Lightweight Bash server monitor — zero dependencies, instant insights

- Tracks **CPU, RAM, and disk usage** with top-5 process reporting
- Pure Shell — no external tools required

---

### 🚀 [logpack-sh](https://github.com/madhawapathum/logpack-sh)
> Bash log management toolkit for Nginx traffic analysis and log archiving

- Analyses Nginx access patterns and automates log rotation/archiving
- Zero-dependency design for portability across Linux environments
